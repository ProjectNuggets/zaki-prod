## What Was Built

**Task 1 (C2):** Imported `SandboxBadge` from `@/app/components/agent/SandboxBadge` into `ChatArea.tsx`. Added `sandboxState` selector (`useZakiSessionUiStore((s) => s.sandbox)`). Mounted `<SandboxBadge active={isZakiBotActiveSpace} sandbox={sandboxState} className="ml-2" />` in the breadcrumb header row between the space/thread label and the action buttons. Badge is self-hiding when `!active || !sandbox || !sandbox.enabled`.

**Task 2 (H7):** Added a `useEffect` in `ChatArea.tsx` that registers a global `keydown` listener only when `nullalisApprovalRequest` is non-null. Pressing `y`/`Y` calls `handleApprovalAction(id, true)` (approve); `n`/`N` calls `handleApprovalAction(id, false)` (deny). Guards prevent firing when focus is in `INPUT`, `TEXTAREA`, or `contenteditable`. Listener is cleaned up on unmount or when approval clears.

**Task 3 (H8):** Added `import { useTranslation } from "react-i18next"` and `const { t } = useTranslation()` inside `ApprovalRequiredCard`. Replaced all 9+ hardcoded English strings with `t("zakiControls.approval.*")` calls. Added 11 keys (`title`, `defaultReason`, `riskLabel`, `approveAria`, `denyAria`, `approveBtn`, `denyBtn`, `approvingState`, `denyingState`, `decidedApproved`, `decidedDenied`) to both `en.json` and `ar.json` under `zakiControls.approval`.
