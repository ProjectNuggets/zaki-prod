# Phase 03 Context: Capability Parity Completion

## Objective

Ensure every upstream learning-relevant capability is available in ZAKI as a ZAKI-owned offering.

## Capabilities

- tutor chat
- session history
- source upload/RAG
- document/image/folder/archive upload
- books and lesson reader
- quizzes and attempts
- question bank/review
- notebooks/saved records
- co-writer
- deep research
- deep solve
- visualization
- math animation
- tutorbot/managed agents
- safe skills/tools
- settings split

## Rule

No capability is dropped because it overlaps with another ZAKI feature. Overlaps are separate offerings unless the user explicitly decides otherwise.
