Need help?

Common fixes for access, billing, stream reliability, and memory behavior.

Access code says expired

Redeem a fresh code in the app and refresh the chat view.

Chat stream failed

Retry in the same thread. If repeated, share timestamp and workspace/thread details.

Billing and subscription controls

Manage billing and cancellation from app settings when billing is configured.

Memory item not visible

Open Memory and review pending/conflicts. Report prompt + timestamp if mismatch persists.
